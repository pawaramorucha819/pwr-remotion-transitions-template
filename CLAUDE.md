# Remotion Transitions Dictionary

## 概要
24種類のトランジション表現を Remotion コンポジションとして実装した辞書プロジェクト。
各トランジションはシーンAとシーンBの切り替わり方を視覚的にデモする。

## トランジション一覧

| ID | 日本語名 | 説明 |
|---|---|---|
| Cut | カット | 前後を切り替える最も基本の繋ぎ。瞬時に場面転換する |
| JumpCut | ジャンプカット | 同一ショット内の時間を飛ばしてテンポを上げる |
| MatchCut | マッチカット | 形・動き・構図が似た要素で自然につなぐ |
| SmashCut | スマッシュカット | 静→激など、強いコントラストで唐突に切り替えて驚かせる |
| LCut | Lカット | 映像は先に切り替わるが、音声は前の場面が少し残る |
| JCut | Jカット | 音声が先に次の場面のものが入ってから映像が切り替わる |
| CrossDissolve | クロスディゾルブ | 前の映像が薄れつつ次の映像が重なって現れる |
| FadeIn | フェードイン | 黒から徐々に映像が現れる |
| FadeOut | フェードアウト | 映像が徐々に黒へ消えていく |
| DipToBlack | ディップ・トゥ・ブラック | 一瞬黒へ落としてから次へ |
| DipToWhite | ディップ・トゥ・ホワイト | 一瞬白へフラッシュしてから次へ |
| Wipe | ワイプ | 画面を線が横切り、押し出すように切り替える |
| SlideTransition | スライドトランジション | 画面全体が上下にスライドして次の画面へ |
| Push | プッシュ（押し出し） | 次の画面が押し込んできて前の画面を追い出す |
| Iris | アイリス | 円形マスクが開閉して場面が現れる/消える |
| ZoomTransition | ズームトランジション | ズームインの勢いで次のショットへ繋ぐ |
| WhipPan | ホイップパン | 高速パンのブレを利用して場面転換する |
| SpinTransition | スピントランジション | 画面が回転しながら切り替わる |
| Morph | モーフ | 前後の要素が滑らかに形状変化して次へつながる |
| GlitchTransition | グリッチトランジション | ノイズ/歪み/色ズレで壊れて切り替わる |
| FreezeToCut | フリーズ→カット | 一瞬静止してから次へ切り替える |
| FilmBurn | フィルムバーン | フィルム焼け風のフラッシュで切り替える |
| LightLeak | ライトリーク | 光漏れのオーバーレイで柔らかく場面転換する |
| LensFlareWipe | レンズフレアワイプ | 強いフレアが画面を横切り、切り替わる |

## プロジェクト構造
- `src/categories/01-transitions.tsx` - 全24トランジションコンポーネント
- `src/Root.tsx` - コンポジション登録
- `src/utils/` - 共有ユーティリティ（constants, schema, label, font, noise, commonScenes）
- `public/` - 静的アセット（画像、音声、フォント）
